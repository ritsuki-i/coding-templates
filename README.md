# Code Shelf

検索してコピーできる、依存関係なしのコーディングテンプレート集です。

## 起動

`templates.json` を読み込むため、ローカルサーバー経由で開きます。

```powershell
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

## データ

- 初期データ: `templates.json`
- Webアプリ上で追加・編集・削除したデータ: ブラウザのローカルストレージ
- 画面上部からJSONの読み込み・書き出しが可能

検索欄では半角・全角スペースで単語を区切れます。説明またはコードに含まれる検索語数が多い順、同数なら説明内の一致数が多い順に表示します。
